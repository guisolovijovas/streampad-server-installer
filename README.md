# StreamPad Server Installer

This repository contains the Windows installer for StreamPad Server.

## Download

Use the installer file in:

- `releases/Instalador_StreamPad_Server.exe`

## What this installer does

- Installs StreamPad Server on Windows
- Includes the server executable and startup helpers
- Prepares the local environment for the mobile app connection

## Notes

- Windows only
- You may need firewall permission during first setup
- Run as administrator if your system policy requires it

## Versioning

For each new server release:

1. Replace the `.exe` in `releases/`
2. Commit and push
3. Create a GitHub Release and attach the installer
